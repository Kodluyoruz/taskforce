# Wrapper Sınıflar

Wrapper sınıflar sayesinde, ilkel veri tiplerini (int,byte,sort vs.) nesne (object) olarak kullanmamızı sağlayan sınıflardır.

| **İlkel Veri tipleri** | **Wrapper Sınıfları** |
| ------------ | ------------------------- |
| byte         | Byte                      |
| short        | Short                     |
| int          | Integer                   |
| long         | Long                      |
| float        | Float                     |
| double       | Double                    |
| boolean      | Boolean                   |
| char         | Character                 |
