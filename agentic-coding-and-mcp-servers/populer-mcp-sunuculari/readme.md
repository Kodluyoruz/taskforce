# Popüler MCP Sunucuları ve Kullanım Alanları

MCP ekosistemi hızla büyüyor. Topluluk ve şirketler tarafından geliştirilen hazır sunucuları kullanarak, kod yazmadan da AI asistanınızı güçlendirebilirsiniz. İşte en popüler MCP sunucuları:

## 1. Filesystem (Dosya Sistemi) Sunucusu

- **Ne Yapar?:** Bilgisayarınızdaki belirli klasörlere ve dosyalara AI'ın erişmesini sağlar.
- **Kullanım Alanı:** Proje dosyalarınızı okutmak, logları analiz ettirmek veya yeni kod dosyaları oluşturmak.
- **Örnek:** "Masaüstümdeki 'Raporlar' klasöründeki tüm PDF'leri özetle."

## 2. PostgreSQL Sunucusu

- **Ne Yapar?:** PostgreSQL veritabanınıza bağlanır. Tablo şemalarını okuyabilir ve SQL sorguları çalıştırabilir.
- **Kullanım Alanı:** Veri analizi, raporlama, veritabanı şeması optimizasyonu.
- **Örnek:** "Users tablosunda geçen ay kayıt olan kullanıcıların sayısını getir."

## 3. Git & GitHub Sunucusu

- **Ne Yapar?:** Git repolarını okuyabilir, commit geçmişini inceleyebilir, diff alabilir ve GitHub API'si üzerinden issue/PR oluşturabilir.
- **Kullanım Alanı:** Kod incelemesi (Code Review), sürüm notları (Release Notes) hazırlama, hata takibi.
- **Örnek:** "Son commit'te yapılan değişiklikleri analiz et ve potansiyel hataları listele."

## 4. Google Drive & Google Docs Sunucusu

- **Ne Yapar?:** Google Drive'daki dosyalarınızı arayabilir ve içeriklerini okuyabilir.
- **Kullanım Alanı:** Doküman yönetimi, bilgi bankası taraması.
- **Örnek:** "'Proje X' ile ilgili olan tüm dokümanları bul ve özet bir rapor hazırla."

## 5. Slack Sunucusu

- **Ne Yapar?:** Slack kanallarındaki mesajları okuyabilir ve yeni mesaj gönderebilir.
- **Kullanım Alanı:** Takım içi iletişim analizi, otomatik bildirimler, toplantı özetleri.
- **Örnek:** "#general kanalında bugün konuşulan önemli konuları özetle."

## 6. Brave Search (İnternet Araması) Sunucusu

- **Ne Yapar?:** Brave arama motorunu kullanarak internette arama yapar.
- **Kullanım Alanı:** Güncel bilgilere erişim, araştırma yapma.
- **Örnek:** "React 19 ile gelen yeni özellikleri araştır ve listele."

## Nasıl Kullanılır?

Bu sunucuların çoğu açık kaynaklıdır ve `npx` veya `docker` komutları ile kolayca çalıştırılabilir. Claude Desktop konfigürasyon dosyanıza (`claude_desktop_config.json`) ilgili komutları ekleyerek hemen kullanmaya başlayabilirsiniz.
