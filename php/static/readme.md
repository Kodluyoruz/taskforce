### Static

Sınıf özelliklerinin ya da methodların static olarak tanımlanması, o özelliklerin ve yöntemlerin sınıfı başlatmadan kullanılabilmesini sağlar.
- Static methodlar Ram'de tutulur ve daha performaslıdır.
