PostgreSQL Kurulumu
======
Çalışmalarımız boyunca veritabanı yönetim sistemi olarak PostgreSQL kullanacağız. PostgreSQL açık kaynak kodlu ve popüler bir ilişkisel veritabanı yönetim
sistemidir.

### Neden PostgreSQL?
- Açık kaynak kodlu
- Çeşitli veri tipi desteği
- Güçlü dökümantasyon ve topluluk desteği
- Tüm işletim sistemlerine uygunluk
- ACID uyumluluk
- Güvenlidir
- Büyük verilerle kolay çalışma
- .....
